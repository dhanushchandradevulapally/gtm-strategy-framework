# GTM Strategy Framework

A complete, battle-tested Go-To-Market strategy framework for B2B SaaS products. Built from competitive intelligence analysis of Salesforce ($37.9B), HubSpot ($3.1B), and Freshworks ($839M) using S&P Capital IQ data.

## What's Inside

This repository contains structured templates and frameworks covering every dimension of SaaS GTM strategy:

| Module | Templates | Purpose |
|--------|-----------|---------|
| **01-ICP** | Scoring matrix, persona templates, research checklists | Define and prioritize ideal customers |
| **02-Pricing** | Model comparison, tier templates, unit economics calculator | Design value-aligned pricing |
| **03-Channel** | Funnel metrics, sales playbooks, conversion benchmarks | Execute PLG and sales-led motions |
| **04-Positioning** | Competitive matrix, battlecards, messaging framework | Differentiate in crowded markets |
| **05-Launch** | Timeline templates, milestone trackers, KPI dashboards | Orchestrate product launches |
| **06-Content** | Content calendars, case study templates, ROI calculators | Enable demand generation |

## Quick Start

1. **Fork this repository** for your product
2. **Start with `01-icp/`** -- define your Ideal Customer Profile using the scoring matrix
3. **Move to `02-pricing/`** -- model your pricing strategy with the unit economics calculator
4. **Use `04-positioning/`** -- build competitive battlecards for your sales team
5. **Track progress in `05-launch/`** -- manage your GTM timeline and KPIs

## Framework Philosophy

This framework is built on three principles derived from public SaaS company analysis:

1. **Hybrid GTM wins** -- HubSpot's PLG + sales-assist model delivers 19%+ CAGR at 83%+ gross margins
2. **Usage-based pricing scales** -- Aligns customer value with revenue; enables land-and-expand
3. **Vertical depth beats horizontal breadth** -- Start narrow, dominate one segment, then expand
